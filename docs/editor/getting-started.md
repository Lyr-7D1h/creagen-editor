# Getting Started

Welcome to Creagen Editor, a creative coding web editor focused on being minimal, customizable, fast, and powerful.

Its goal is to easily make and share creative coding projects.

## Run and Render

### Desktop

1. Write code.
2. Press **Control+Shift+Enter** to render code, or press the top-right render button.
3. Open the **Menu** (sideways arrow in the top-left corner) to manage dependencies, saved changes, keybindings, settings, and more.

### Mobile

1. To run the current code, tap the button in the top-right corner.
2. Open the **Menu** to view saved changes, settings, and other project options.
